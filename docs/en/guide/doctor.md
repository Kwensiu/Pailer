# Doctor - System Health

The Doctor page is your diagnostic center for maintaining Scoop system health. It provides comprehensive system checks, problem diagnosis, and repair functions to help you keep your system in optimal condition.

## Page Overview

The Doctor page contains multiple diagnostic and maintenance modules:

- **System Check**: Comprehensive check of Scoop configuration and environment
- **Cache Management**: Clean and manage cache files
- **Versioned Apps Management**: Manage multi-version applications
- **Shim Management**: Manage command-line shortcuts
- **System Information**: Display detailed system status information

## Basic Usage

### 1. System Check

System check automatically detects common problems:

- **Environment Configuration**: Check if Scoop environment is correctly configured
- **Path Settings**: Verify system paths and environment variables
- **Permission Check**: Check file and directory permissions
- **Dependency Verification**: Verify necessary dependencies are installed

### 2. View Check Results

Check results are displayed clearly:

- **Status Indicators**: Use colors and icons to show problem severity
- **Problem Description**: Detailed description of discovered problems
- **Solutions**: Provide specific repair suggestions
- **One-Click Fix**: Some problems support one-click automatic repair

### 3. Run Diagnostics

Manually run various diagnostics:

- **Full Check**: Run comprehensive system diagnostics
- **Specific Check**: Only check specific aspects
- **Re-check**: Re-verify problems after repairs

## Advanced Features

### Cache Management

Cache management helps optimize system performance:

- **View Cache**: Display size and status of various caches
- **Clean Cache**: Safely clean expired cache files
- **Auto Clean**: Set automatic cleaning rules
- **Space Analysis**: Analyze disk space usage by cache

### Versioned Apps Management

Manage applications with multiple versions in the system:

- **Version List**: Display all multi-version applications
- **Version Cleanup**: Clean unwanted old versions
- **Version Switching**: Quickly switch between versions
- **Dependency Analysis**: Analyze dependencies between versions

### Shim Management

Manage command-line shortcuts:

- **Shim List**: Display all available commands
- **Repair Shim**: Repair damaged shortcuts
- **Rebuild Shim**: Regenerate shortcuts
- **Path Check**: Verify validity of Shims

### Helper Tools

Install and manage system helper tools:

- **Tool Detection**: Detect missing helper tools
- **One-Click Install**: Automatically install necessary tools
- **Version Management**: Manage tool versions
- **Configuration Check**: Verify tool configurations

## Common Issues

### What to do when check finds problems?

1. **View Details**: Read problem descriptions and causes carefully
2. **Follow Suggestions**: Operate according to provided solutions
3. **Re-check**: Re-run checks after repairs to verify
4. **Seek Help**: If problems persist, seek community help

### Is cache cleaning safe?

1. **Auto Cleaning is Safe**: System automatically identifies safely cleanable cache
2. **Manual Cleaning with Caution**: Confirm content carefully when cleaning manually
3. **Backup Important Data**: Backup important cache before cleaning
4. **Batch Cleaning**: Recommend cleaning large cache in batches

### How to fix damaged Shims?

1. **Auto Repair**: Use auto repair function in Shim management
2. **Reinstall**: Reinstall related software
3. **Manual Rebuild**: Manually rebuild damaged Shims
4. **Check Permissions**: Ensure sufficient permissions for operations
