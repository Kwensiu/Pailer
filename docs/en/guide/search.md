# Search - Discover Software

The Search page is your starting point for exploring and discovering software within your installed repositories. It searches through all repositories you have locally added, not online search.

## Page Overview

The Search page is located in the main navigation bar. Click "Search" to access it. The page mainly includes:

- **Search Bar**: Enter keywords to search for software in local repositories
- **Refresh Button**: Refresh current search results
- **Result Category Tabs**: Distinguish between packages and binaries
- **Bucket Filter**: Filter search results by local repositories
- **Paginated Results List**: Display search results, 8 results per page

## Basic Usage

### 1. Search for Software

Enter keywords in the search bar, and the system will search within your locally installed repositories:

- **Local Search**: Only searches repositories you have added to the system
- **Search by Name**: Enter partial or complete characters of the software name
- **Real-time Search**: Search as you type, no need to press Enter
- **Cache Support**: Search results are cached to improve repeated search speed

### 2. View Search Results

Search results are divided into two tabs:

- **Packages**: Complete applications with installation manifests
- **Binaries**: Standalone executable files, usually smaller

### 3. Filter Results

Use the bucket filter at the top to:

- **View All Buckets**: Display search results from all installed repositories
- **Filter Specific Buckets**: Show only results from specified local repositories
- **Quick Location**: Quickly find software from specific sources

### 4. Paginated Browsing

Search results support paginated display:

- **8 Results Per Page**: Avoid loading too much content at once
- **Pagination Controls**: Bottom pagination controls support page jumping
- **Keyboard Navigation**: Use left/right arrow keys to quickly switch pages

## Important Notes

### Search Scope Limitations

- **Local Repositories Only**: Search function only covers repositories you have installed
- **Not Online Search**: Does not search the internet for software
- **Depends on Repository Management**: The richness of search results depends on the number of repositories you have added

### Expanding Search Scope

If you can't find the software you need:

1. **Add More Repositories**: Add repositories containing that software in the Bucket page
2. **Update Existing Repositories**: Ensure repositories are updated to the latest version
3. **Check Official Repositories**: Ensure you have added official repositories like main, extras

## Advanced Features

### View Software Details

Click on software in search results to view:

- **Version Information**: Currently available versions
- **Software Description**: Function description and purpose
- **Homepage Link**: Software's official website
- **Repository Information**: Source repository and branch information

### View Installation Manifest

For advanced users, you can view the software's installation manifest:

- **Manifest Viewing**: Click version number to view JSON format installation configuration
- **Dependencies**: View software dependencies and installation requirements
- **Installation Options**: Understand configurable installation parameters

### View Repository Information

Click on the source repository name to:

- **Repository Details**: View detailed repository information
- **Application List**: Browse all applications in the repository
- **Repository Status**: Understand repository update status

### Refresh Search Results

Use the refresh button next to the search bar:

- **Re-search**: Re-execute current search
- **Clear Cache**: Clear search cache to get latest results
- **Update Status**: Refresh button rotates when loading

## Usage Tips

### Search Techniques

- **Use Keywords**: Enter keywords related to software functionality
- **Try English**: Many software use English names
- **Partial Matching**: No need to enter complete names

### Result Filtering

- **Filter by Repository**: If you know the software source, you can filter quickly
- **View Binaries**: Simple tools might be in the binary category
- **Compare Versions**: Compare the same software from different repositories

### Keyboard Operations

- **Arrow Keys**: Quickly switch pagination in search results
- **Avoid Conflicts**: Arrow keys are used for text editing when in input fields

## Common Issues

### Can't find software?

1. **Check Keywords**: Try different keywords or English original names
2. **Add More Repositories**: You might need to add repositories containing that software
3. **Update Repositories**: Ensure software repositories are updated to the latest version
4. **Check Categories**: Look in the binary files tab

### Few search results?

1. **Insufficient Repositories**: Add more official and community repositories
2. **Repositories Not Updated**: Update all repositories to get latest software list
3. **Keywords Too Specific**: Try more general keywords

### Installation failed?

1. **View Error Messages**: Read error prompts carefully during installation
2. **Check Dependencies**: Ensure system meets software dependency requirements
3. **Retry Installation**: Network issues might cause installation failure

### How to find latest version?

1. **View Version Information**: Check available versions in software details
2. **Update Repositories**: Ensure repository information is up-to-date
3. **Switch Repositories**: Different repositories might have different versions

### Pagination display issues?

1. **Refresh Results**: Click refresh button to reload
2. **Clear Cache**: Refresh clears search cache
3. **Check Filters**: Ensure repository filter settings are correct

## Related Features

The Search page is closely integrated with other functions:

- **Bucket Page**: Manage software source repositories to expand search scope
- **Installed Page**: After installation, you can view and manage on the installed page
- **Settings Page**: Configure search-related settings

Through the Search page, you can quickly discover and install needed software within your installed repositories. To get richer search results, ensure you have added sufficient software repositories.
