# 快速开始

本指南将帮助您在几分钟内完成 Pailer 的安装、配置和开始使用。

## 1. 安装 Pailer

### 方法一：GitHub Releases（推荐）

1. 访问 [Releases 页面](https://github.com/Kwensiu/Pailer/releases)
2. 下载最新的 `.nsis` 安装程序或 `Portable` 便携版
3. 运行安装程序并按照提示完成安装

### 方法二：通过 Scoop 安装

如果您已经安装了 Scoop，可以通过以下命令安装 Pailer：

```powershell
scoop bucket add carrot https://github.com/Kwensiu/scoop-carrot
scoop install pailer
```

## 2. 首次启动和配置

### 2.1 启动应用

安装完成后，启动 Pailer 应用。

### 2.2 Scoop 配置向导

首次启动时，如果系统未检测到 Scoop 配置，会自动弹出 **Scoop 配置向导**：

1. **自动检测**：系统会尝试自动检测现有的 Scoop 安装
2. **手动配置**：如果自动检测失败，您可以手动指定 Scoop 路径：
   - 默认路径：`C:\Users\{用户名}\scoop`
   - 自定义路径：选择您的 Scoop 安装目录
3. **验证配置**：点击验证按钮确认配置正确
4. **完成配置**：验证通过后，点击完成进入主界面

### 2.3 初始化数据加载

配置完成后，系统会自动加载初始数据：
- 已安装软件列表
- 可用软件仓库
- 系统状态信息

这个过程通常需要几秒钟到一分钟，取决于您的软件数量。

## 3. 基本使用流程

### 3.1 搜索和安装软件

1. **打开搜索页面**：点击导航栏的"搜索"
2. **输入关键词**：在搜索框中输入软件名称，如 "vscode"
3. **查看结果**：浏览搜索结果，区分软件包和二进制文件
4. **查看详情**：点击软件查看版本、描述等详细信息
5. **安装软件**：点击"安装"按钮开始安装
6. **等待完成**：观察安装进度，等待安装完成

### 3.2 管理已安装软件

1. **打开已安装页面**：点击导航栏的"已安装"
2. **查看软件列表**：浏览所有已安装的软件
3. **检查更新**：页面顶部会显示可更新的软件数量
4. **更新软件**：点击"更新"按钮或使用"全部更新"
5. **卸载软件**：点击软件的操作菜单选择"卸载"

### 3.3 管理软件仓库

1. **打开仓库页面**：点击导航栏的"仓库"
2. **查看现有仓库**：浏览已安装的仓库列表
3. **更新仓库**：点击"更新"按钮获取最新软件列表
4. **搜索新仓库**：使用搜索功能发现新的软件源

## 4. 常见设置

### 4.1 界面个性化

进入 **设置页面** 进行个性化配置：

- **主题设置**：选择浅色、深色或跟随系统
- **语言设置**：切换中文/英文界面
- **启动页面**：设置默认启动页面

### 4.2 自动化配置

- **自动更新**：设置自动检查软件更新的频率
- **自动清理**：配置定期清理缓存文件

## 5. 遇到问题？

### 5.1 常见问题排查

1. **Scoop 配置问题**：重新运行配置向导
2. **网络连接问题**：检查网络设置和代理配置
3. **权限问题**：确保有足够的管理员权限
