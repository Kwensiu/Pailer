# Packages - Installed

The Installed page is your control center for managing all software on your system. Here, you can view, update, uninstall, and manage all software installed through Scoop.

## Page Overview

The Installed page provides comprehensive software management features:

- **Software List**: Display all installed software and their status
- **Update Check**: Automatically check and display updatable software
- **Batch Operations**: Support batch updates and uninstalls
- **Version Management**: Manage multiple versions of software
- **Status Control**: Lock/unlock software versions

## Basic Usage

### 1. View Installed Software

The page displays all installed software by default:

- **Software Name**: Identifier of the software
- **Current Version**: Currently installed version number
- **Source Bucket**: Which bucket the software comes from
- **Update Status**: Whether updates are available
- **Installation Time**: Installation time of the software

### 2. Check for Updates

The system automatically checks for software updates:

- **Update Count**: Display count of updatable software at the top
- **Update Button**: Individual update button for each software
- **Update All**: Batch update all updatable software
- **Progress Display**: Real-time display of update progress

### 3. Manage Software

Perform various management operations on installed software:

- **Uninstall Software**: Remove unwanted software
- **View Information**: View detailed software information
- **Open Folder**: Quick access to software installation directory
- **Switch Versions**: Switch between multiple versions

## Advanced Features

### Version Management

For software supporting multiple versions, you can manage versions:

- **View All Versions**: Display all available versions of software
- **Install Specific Version**: Install specified historical version
- **Version Switching**: Switch between installed versions
- **Version Cleanup**: Clean up unwanted old versions

### Status Control

Control update status of software:

- **Lock Version**: Prevent automatic software updates
- **Unlock Version**: Allow normal software updates
- **View Status**: Display lock status of software
- **Batch Operations**: Batch lock or unlock software

### Filtering and Sorting

Use powerful filtering and sorting features:

- **Filter by Name**: Quickly find software by name
- **Filter by Bucket**: Display only software from specific buckets
- **Filter by Version Type**: Distinguish between stable and development versions
- **Filter by Update Status**: Display only software needing updates

### View Modes

Support different display modes:

- **List View**: Detailed table display, suitable for large amounts of software
- **Grid View**: Card-style display, more intuitive
- **Compact View**: Space-saving display method

## Usage Tips

### Update Strategy

- **Selective Updates**: Only update needed software to avoid compatibility issues
- **Batch Updates**: Use batch updates to improve efficiency
- **Pre-Update Check**: View update logs to understand changes

### Version Management

- **Keep Stable Versions**: Keep current stable versions before updating
- **Test New Versions**: Test new versions before full update
- **Clean Old Versions**: Regularly clean unwanted old versions

### Performance Optimization

- **Regular Cleanup**: Uninstall software you no longer need
- **Monitor Size**: Pay attention to disk space usage by software
- **Optimize Startup**: Manage auto-start software

## Common Issues

### Update failed?

1. **Check Network Connection**: Ensure network connection is normal
2. **View Error Messages**: Read error prompts carefully
3. **Restart Update**: Try restarting the update process
4. **Check Dependencies**: Ensure system meets update dependencies

### Can't uninstall software?

1. **Check Permissions**: Ensure you have sufficient permissions to uninstall
2. **Close Programs**: Ensure software is not running
3. **Check Dependencies**: Other software might depend on this software
4. **Force Uninstall**: Use force uninstall option

### How to resolve version conflicts?

1. **View Conflict Details**: Understand the specific conflict reasons
2. **Uninstall Conflicting Versions**: Remove conflicting versions
3. **Reinstall**: Reinstall needed versions
4. **Clear Cache**: Clear possible cache issues

## Best Practices

### Software Management Strategy

- **Regular Update Checks**: Keep software up-to-date
- **Backup Important Configurations**: Backup important configurations before updating
- **Test Environment**: Test updates in test environment first

### System Maintenance

- **Regular Cleanup**: Uninstall software you no longer need
- **Monitor Disk Space**: Pay attention to storage space used by software
- **Performance Monitoring**: Pay attention to software impact on system performance

## Related Features

The Installed page is closely integrated with other pages:

- **Search Page**: Install new software from the search page
- **Bucket Page**: Manage source buckets of software
- **Doctor Page**: Check and fix software-related issues
- **Settings Page**: Configure software management related settings

Through the Installed page, you can comprehensively manage software on your system, ensuring the software environment remains in optimal condition.
