# 软件包 - 已安装

已安装页面是您管理系统中所有软件的控制中心。在这里，您可以查看、更新、卸载和管理所有通过 Scoop 安装的软件。

## 页面概览

已安装页面提供了全面的软件管理功能：

- **软件列表**：显示所有已安装软件及其状态
- **更新检查**：自动检查并显示可更新的软件
- **批量操作**：支持批量更新和卸载
- **版本控制**：锁定/解锁软件版本，管理版本化应用
- **视图切换**：支持列表视图和网格视图

## 功能特性

### 软件显示

- **软件名称**：软件的标识符
- **当前版本**：当前安装的版本号
- **来源仓库**：软件来自哪个仓库
- **更新状态**：是否有可用更新
- **安装时间**：软件的安装时间（在详细信息中显示）
- **锁定状态**：软件是否被锁定防止更新

### 更新管理

- **更新计数**：顶部显示可更新的软件数量
- **单个更新**：单个软件的更新按钮
- **批量更新**：批量更新所有可更新软件
- **进度显示**：实时显示更新进度

### 软件操作

- **卸载软件**：移除不需要的软件
- **查看信息**：查看软件的详细信息
- **打开文件夹**：快速访问软件安装目录
- **更改仓库**：更改软件的来源仓库

## 高级使用

### 版本管理

- **版本切换**：在已安装的不同版本间切换
- **版本删除**：删除不需要的旧版本
- **版本信息**：查看所有可用版本和当前版本
- **安装时间**：显示当前版本的安装时间

### 版本控制

- **锁定版本**：防止软件自动更新
- **解锁版本**：允许软件正常更新
- **查看状态**：显示软件的锁定状态
- **批量操作**：批量锁定或解锁软件

### 视图与筛选

- **列表视图**：详细的表格显示，适合大量软件
- **网格视图**：卡片式显示，更加直观
- **视图切换**：点击按钮快速切换视图模式
- **按名称筛选**：根据软件名称快速查找
- **按仓库筛选**：只显示特定仓库的软件
- **按版本类型筛选**：区分全部、版本化、锁定的软件
- **按更新状态筛选**：只显示需要更新的软件

### 系统状态

- **Scoop状态检查**：显示Scoop的整体状态
- **问题应用**：列出有问题的应用
- **更新提醒**：提醒Scoop或仓库需要更新
- **网络状态**：检查网络连接状态

### 仓库管理

- **仓库选择**：选择新的目标仓库
- **版本对比**：对比不同仓库中的版本
- **迁移安装**：将软件迁移到新仓库

## 相关功能

已安装页面与其他功能紧密集成：

- **搜索页面**：可以从搜索页面安装新软件
- **仓库页面**：管理软件来源的仓库
- **医生页面**：检查和修复软件相关的问题
- **设置页面**：配置软件管理的相关设置

通过已安装页面，您可以全面管理系统中的软件，确保软件环境保持最佳状态。
