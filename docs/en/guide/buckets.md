# Buckets - Manage Sources

The Bucket page is your control center for managing Scoop software sources. Through this page, you can add, update, and manage all software buckets to ensure access to the latest software.

## Page Overview

The Bucket page provides complete bucket management functionality:

- **Bucket List**: Display all installed buckets and their status
- **Search Function**: Discover and add new community buckets
- **Batch Operations**: Update multiple buckets simultaneously
- **Detailed Information**: View detailed information and statistics of buckets

## Basic Usage

### 1. View Installed Buckets

The page displays all installed buckets by default:

- **Bucket Name**: Unique identifier of the bucket
- **Update Status**: Shows whether the bucket needs updating
- **App Count**: Number of software packages in the bucket
- **Last Update**: Last update time of the bucket
- **Git Information**: Git branch and remote address of the bucket

### 2. Update Buckets

Keeping buckets updated is key to getting the latest software:

- **Single Update**: Click the update button on the bucket card
- **Batch Update**: Use the "Update All" button at the top
- **Progress Display**: Real-time display of update progress and results

### 3. Search New Buckets

Discover new software sources through the search function:

- **Keyword Search**: Enter related keywords to search buckets
- **Category Browsing**: Browse popular buckets by category
- **Community Recommendations**: View community-recommended bucket lists

## Advanced Features

### Bucket Details

Click on a bucket to view detailed information:

- **Bucket Description**: Purpose and features of the bucket
- **App List**: All software packages in the bucket
- **Statistics**: App count, download statistics, etc.
- **Maintenance Info**: Maintainer and update frequency

### Branch Management

Advanced users can manage bucket branches:

- **View Branches**: Display available branches of the bucket
- **Switch Branches**: Switch to different branch versions
- **Branch Description**: Understand the purpose of different branches

### Bucket Installation

Install new buckets from search results:

- **One-Click Install**: Click install button to add bucket
- **Installation Confirmation**: Confirm bucket info and installation options
- **Progress Tracking**: Real-time display of installation progress

## Usage Tips

### Choose Appropriate Buckets

- **Official Buckets**: `main` bucket contains the most stable software
- **Extras Bucket**: Contains more utility tools and applications
- **Versions Bucket**: Like `versions` bucket contains multiple versions of software
- **Community Buckets**: Add specific community buckets as needed

### Maintain Bucket Health

- **Regular Updates**: Keep buckets regularly updated to get latest software
- **Clean Unused Buckets**: Remove buckets you no longer need
- **Monitor Updates**: Pay attention to update status of important buckets

### Batch Operation Optimization

- **Selective Updates**: Only update needed buckets to improve efficiency
- **Off-Peak Updates**: Avoid batch updates during network peak hours
- **Error Handling**: Pay attention to failed buckets and retry

## Common Issues

### Bucket update failed?

1. **Check Network Connection**: Ensure network connection is normal
2. **Git Configuration**: Check if Git configuration is correct
3. **Bucket Address**: Confirm bucket address is valid
4. **Permission Issues**: Check if you have sufficient permissions to access buckets

### Can't find needed software?

1. **Add More Buckets**: Add buckets containing that software
2. **Search Community Buckets**: Search in community buckets
3. **Check Software Name**: Confirm the correct name of the software

### Bucket conflicts?

1. **Remove Duplicate Buckets**: Avoid adding duplicate buckets
2. **Check Priority**: Understand bucket priority settings
3. **Clear Cache**: Clear possible cache conflicts

## Best Practices

### Bucket Management Strategy

- **Minimalism Principle**: Only add needed buckets, avoid too many
- **Regular Maintenance**: Regularly check and update buckets
- **Backup Configuration**: Backup important bucket configurations

### Performance Optimization

- **Control Quantity**: Avoid adding too many buckets affecting performance
- **Update Frequency**: Set reasonable update frequency
- **Cache Management**: Regularly clean expired cache data

## Related Features

The Bucket page is closely related to other functions:

- **Search Page**: Search results depend on installed buckets
- **Installed Page**: Display installed software from various buckets
- **Settings Page**: Configure bucket-related settings

Through the Bucket page, you can effectively manage software sources, ensuring access to rich and latest software resources.
